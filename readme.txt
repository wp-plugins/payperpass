Monétisez votre site avec la solution de micropaiement payperpass. Simple, sécurisé et gratuit, sans connaissances techniques particulières.
Votre client est débité sur sa facture téléphonique.

== Description ==

Ce plugin vous permet d'ajouter facilement un iframe de paiement.
Pour cela, installer le plugin, et rendez-vous dans votre [Espace Client](http://client.payperpass.eu/espaceClient/ficheclient.aspx) pour générer votre api key.

Ensuite, créez vos shortcodes facilement grâce à la fenêtre *Payperpass* dans l'editeur de votre page.

Le shortcode s'ajoutera automatiquement à votre texte :
> [payperpass idProduit=1 idService=1]

== Installation ==

Vous êtes déjà client chez *Payperpass*, ajoutez l'extension à votre site, activez le plugin puis saisisez vos shortcodes dans vos pages.

Vous n'êtes pas encore client,
1. Créez votre compte sur [Payperpass](http://client.payperpass.eu/inscription.aspx)
2. [Ajoutez votre site](http://client.payperpass.eu/espaceClient/sites.aspx)
3. [Ajouter les produits](http://client.payperpass.eu/espaceClient/produits.aspx) que vous souhaitez monétiser
4. Ajouter le shortcode payperpass pour wordpress
5. Invitez vos clients à payer pour ce produit

== Frequently Asked Questions ==

= Comment suivre l’évolution de mes futurs reversements ? =

Chaque éditeur/webmaster dispose d’un [espace client Payperpass](http://client.payperpass.eu/)
Il peut y suivre les statistiques en temps réel.

= Y a-t-il des frais de mise en service ou d’utilisation des services Payperpass ? =

L’inscription Payperpass est gratuite.
Les reversements et autres virements sont pris en charge par Payperpass.
Aucun frais d’utilisation.

= Dois-je être calé en informatique pour installer les services Payperpass ? =

Nul besoin d’être un expert en informatique pour installer le script de paiement Payperpass.
Après inscription, il vous suffit de télécharger ce dernier dans votre espace client Payperpass et de l’installer.
Un suivi conseil peut être fourni si besoin est.

= J'ai un message d'erreur lors de la =

Vérifiez que votre API KEY a bien été enregistrée dans les paramètres de Payperpass.
