Gestion de vos iframes produit via shortcode

== Description ==

Ce plugin vous permet d'ajouter facilement un iframe de paiement ainsi que le script de sécurité sur vos pages produits.
Pour cela, installer le plugin, et rendez-vous dans votre [Espace Client](http://payperpass.eu/ficheclient.aspx) pour générer votre api key.

Ensuite, créez vos shortcodes facilement grâce à la fenêtre *Payperpass* dans l'editeur de votre page.

Le shortcode s'ajoutera automatiquement à votre texte.
> [payperpass idProduit=1 idService=1]

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `payperpass` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Insert your script in pages or article whith the payperpass shortcode
